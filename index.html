最主要的问题-想想读者

写论文的主要问题不在于你想说什么，你做了什么，或者你有多努力。

问题是你希望读者应该从阅读你的论文中学到什么，特别是对你的研究区域不了解也不感兴趣的。

另一个问题-想想期刊

总的来说有三种期刊，它们的要求与写法各不相同

  • 综合顶级期刊， nature，science，NC，SA等
    • 一句话：研究内容特别是结果/结论是全球关注的（包括不同专业读者）

顶刊是聚焦全球，着重交叉，更类似于科普文章

  • 专业顶级期刊，一般的中科院二区以上以及JCR Q1期刊

    • 一句话：一般要求理论或者方法的创新与改进

  • 专业普通期刊：一般SCI或者EI

    • 一句话：合理的方法、充足的资料、流畅的表达得出的正确且有用的结果

普刊最重要的是方法合理，结果正确，语句通顺

第三个问题：想想文章

  • Scientific theory innovation? 机理研究、理论创新
  • Method advancement or model improvement? 方法、模型的创新或改进
  • Result has important reference value or filling existing gaps? 结果有重要意义或填补某些空白

The basic structure of SCI paper

  1. Abstract
  2. Introduction
  3. Methodology/models
  4. Study area and data
  5. Result
  6. Discussions
  7. Conclusions
  8. Reference
